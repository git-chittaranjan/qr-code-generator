## Packages Installed
- npm install html-to-image
- npm install react-qr-code
- npm install qrcode
- npm install pdf-lib


## This application have three different ways to generate QR Code
1. Simple QR Code Generator with react-qr-code library - Commented in App.jsx
2. QR Code Generator with qrcode library and PDF download option with pdf-lib library - Commmented in App.jsx
3. QR Code Generator with input field library used - react-qr-code, pdf-lib and html-to-image - In used
